This is a procedure for handling outages of electrical power in the server room.

Electrical Power is Off
  1. Do not shut down any UPS or server manually.
  2. Silence the fire alarm.
Electrical Power is On
  1. Ensure SAP-mail is established through the firewall.
  2. Servers should be operational.
    • Ping the IP from the VM.
    • Connect remotely to the server and restart.
    • Check the VM again.
Check and Re-check Application Servers
  • ERP Beyond Login:
    • If not operational, restart the pbaza server.
    • Log in to the PowerMon server and start manually.
    • Ping OMS and Origami servers.
